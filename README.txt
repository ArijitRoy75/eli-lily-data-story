Covid19 Charts


Points Achieved:-
1) Use of Web API to fetch Covid-19 Details
2) Implement a Clickable India map using D3.js
3) Implement a Table with <th><td><tr> and show all fetched data.

Steps to Run the Application:-
1) Extract the files
2) Copy the path of the "index.html" file and paste it on the address bar of you browser.

Assumptions:-
1) Stable Internet Connectivity (Expect delay in the formation of the Map and Table) 
2) Any Modern Web browser
